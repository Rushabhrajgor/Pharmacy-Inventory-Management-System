# Pharmacy-Inventory-Management-System

Project title:
Inventory-Management-System

Short Discription:
This project analyzes traffic data to study vehicle movement patterns, accident distribution, and congestion trends across multiple road locations. The report includes location-wise traffic summaries and hour-wise analysis to identify peak traffic periods and high-density zones. Key insights such as average hourly traffic and peak usage times help understand urban traffic behavior. The project demonstrates data analysis and reporting skills by transforming raw traffic data into meaningful insights for traffic monitoring and planning.

Tools & Technologies:
SQL (MySQL) – Database design and querying,
Microsoft Excel – Dashboard creation and visualization,
Data Modeling – Table relationships and normalization,
Data Analysis – Business insight generation.

Key Features:
Relational database design using SQL,
Inventory and sales management system,
Advanced SQL queries for business analysis,
Low-stock and expiry monitoring,
Monthly sales and customer analytics,
SQL data connected to Excel dashboard for visualization.
